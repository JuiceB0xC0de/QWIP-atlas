# Code of Conduct

This repository follows a pragmatic, low-drama standard of conduct:

- Be direct and factual.
- Keep criticism scoped to the code or design decision.
- Do not harass, threaten, or degrade other contributors.
- Assume good intent unless the evidence says otherwise.
- When something is wrong, point to the file, line, or behavior and suggest a fix.

This is a small working repo, not a forum for performance theater. Keep it useful.
